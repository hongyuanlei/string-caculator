# NodeJS kata start kit

## VScode setup

- Install Jest Runner

Shortcuts: click File -> Preferences -> Keyboard Shortcuts -> "{}" (top right) the json config file will open add this:

```
{
  "key": "alt+1",
  "command": "extension.runJest"
},
{
  "key": "alt+2",
  "command": "extension.debugJest"
},
```
